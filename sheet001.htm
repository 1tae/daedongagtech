아래 정리한 데이터 형태들은 현재 만나 HUB에서 만나의  AWS 서버로 전송하는 데이터의 형태입니다. 대동에 이대로 전송할 것은 아닙니다. 필드항목중 대동에 필요한 데이터만 전송하는 형태로 펌웨어를 간략하게 수정할 예정입니다.
아래 형태는 참고만 하시면 됩니다. (대동/메가존클라우드 측에서는 필요한 필드만 골라주시면됩니다., 재차 말씀드린것처럼 결국 저희가 대동플랫폼으로 데이터 전달을 하기위한 함수들을 추가로 개발할예정입니다. )
